# App catalog builder

Wave 4 establishes the catalog layer above canonical puzzle assets.

This package now owns:

- library manifest construction
- aisle manifest construction
- catalog manifest construction
- search index generation
- compact app export packaging

Current Wave 4 scope:
- classic 9x9 library only
- aisle manifests are built from puzzle records
- book summaries are inferred from puzzle records
- app export is a compact JSON bundle for later Android consumption
