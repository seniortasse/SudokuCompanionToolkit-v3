# Android library domain

Wave 7 establishes the first Android-side catalog consumption layer.

This package now owns:

- catalog manifest models
- aisle/book/puzzle catalog models
- asset loading from `assets/sudoku_library/classic9`
- catalog, book, and puzzle repositories
- catalog puzzle launch adapter

Current Wave 7 scope:
- local asset-backed catalog only
- no conductor integration yet
- no route/phase integration yet
- no assistant intent handling yet

Those belong to later waves.
